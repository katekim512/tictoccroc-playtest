## 📖 소개

MBTI식 유형 테스트의 익숙함을 빌려 **"우리 아이는 어떤 놀이 악어일까?"** 라는 질문으로 부모의 호기심을 자극합니다.
결과는 귀여운 **악어 캐릭터 8종**으로 나오고, 각 유형은 째깍악어의 **원정대·프로그램**과 연결되어 재미가 자연스럽게 전환으로 이어집니다.

- 🎯 **목표** — 부담 없는 재미 콘텐츠로 부모 유입 → 아이 성향에 맞는 프로그램 추천 → 공유를 통한 재유입
- 👶 **타깃** — 3~10세 아이를 둔 부모 (로그인 없이 바로 시작)
- 📈 **핵심 지표** — 테스트 완료율 · 추천 클릭률 · 공유 재유입(바이럴 루프)

## ✨ 주요 기능

- **9문항 진단** — 3개 축 × 각 3문항, 다수결로 8가지 유형 판정
- **이름 개인화** — 아이 이름을 결과 전반에 사용 (한글 조사 자동 처리: "서준은 / 코코는")
- **나이 휠 선택** — 아이폰 알람식 세로 휠 피커
- **맞춤 추천** — 유형별 원정대에 맞는 프로그램 추천 (자체 프로그램 `째깍 PICK` 우선 노출)
- **카카오톡 공유** — 결과를 자랑하며 친구를 테스트로 초대하는 바이럴 카드
- **모바일 최적화** — 전 폭 반응형, iOS 렌더링 이슈 대응

## 🧩 진단 설계

아이의 놀이 성향을 **3가지 축**으로 보고, 각 축마다 3문항(총 9문항)에 답합니다. 축별 **다수결(2/3)** 로 성향을 정해 조합하면 유형이 나옵니다.

| 축             | 질문 주제                            | 양극                                |
| -------------- | ------------------------------------ | ----------------------------------- |
| ⚡ 에너지      | 몸으로 노는가, 앉아서 노는가         | **활동적(A)** ↔ **차분(B)**         |
| 🔍 몰입 스타일 | 원리를 파는가, 이야기를 짓는가       | **탐구·실제(X)** ↔ **상상·서사(Y)** |
| 🧭 노는 방식   | 계획대로 완성하는가, 자유롭게 뻗는가 | **계획·완성(P)** ↔ **자유(F)**      |

> 2 × 2 × 2 = **8가지 놀이 유형**

## 🐊 8가지 놀이 유형

| 유형  | 캐릭터                 | 한 줄 성향             | 추천 원정대 |
| :---: | ---------------------- | ---------------------- | :---------: |
| `AXF` | 찰방 · 수달 🦦         | 활동적인 자유 탐구가   |    과학     |
| `AXP` | 토리 · 다람쥐 🐿️       | 활동적인 실험 완성가   |    과학     |
| `AYF` | 골디 · 골든리트리버 🐕 | 활동적인 이야기 모험가 |    농장     |
| `AYP` | 코코 · 코끼리 🐘       | 활동적인 서사 완주자   |    역사     |
| `BXF` | 뽀롱 · 고슴도치 🦔     | 차분한 탐구 발명가     |     숲      |
| `BXP` | 등이 · 거북이 🐢       | 차분한 분석 연구가     |    역사     |
| `BYF` | 몽실 · 토끼 🐰         | 차분한 상상 예술가     |     숲      |
| `BYP` | 밤이 · 부엉이 🦉       | 차분한 서사 기록가     |    농장     |

## 🗺️ 사용자 흐름

```
인트로  →  프로필(이름·나이)  →  질문 9개  →  분석 로딩  →  결과 + 추천  →  공유
```

| 단계       | 설명                                                             |
| ---------- | ---------------------------------------------------------------- |
| **인트로** | "우리 아이는 어떤 놀이 악어일까요?" 후킹 · 로그인 없이 시작      |
| **프로필** | 아이 이름(개인화) + 나이(휠 선택) 입력                           |
| **질문**   | 1화면 1문항, 선택하면 자동으로 다음 (유형 테스트 표준 UX)        |
| **로딩**   | 신뢰감을 주는 짧은 분석 연출                                     |
| **결과**   | 유형 카드(캐릭터·설명·해시태그·좋아할 활동) + 맞춤 프로그램 추천 |
| **공유**   | 결과 자랑 + 테스트 초대 카카오톡 카드                            |
